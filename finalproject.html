This is my final project for CSC106 at the University of Rhode Island. I created a spin-off version of the popular board game "Battleship" using techniques that I learned from the Khan processingJS library. To play the game, open the link down below. Press the start button, and then chose a difficulty. As the difficulty increases, the number of guesses you have to find the ships decreases. Click on a tile to shoot a missile. If you see a circle, you missed and if you see a boat, you hit! Try to shoot all of the boats before you run out of missiles. If you don't shoot all of the boats in the given number of tries, you lose.
